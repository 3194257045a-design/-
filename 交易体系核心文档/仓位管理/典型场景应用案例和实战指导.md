```

   - 趋势加仓：上升趋势确认后加仓
   - 回调加仓：技术性回调时分批加仓
   - 风险控制：设置滚动止损

4. 收益管理：
   - 分批止盈：达到目标收益分批止盈
   - 移动止损：跟随价格上涨上调止损
   - 资金管理：盈利部分用于再投资

### 4. 牛市风险控制

```
牛市风险特征：
1. 过度追高：价格上涨过快，估值过高
2. 情绪化操作：贪婪心理，重仓操作
3. 调整忽视：忽视回调风险，缺乏止损
4. 错过机会：过于保守，错过上涨机会

牛市风险控制策略：
1. 估值控制：
   - 设置估值上限：PE > 25倍时谨慎
   - 估值分批：高估值时分批减仓
   - 价值回归：估值过高时理性看待

2. 情绪管理：
   - 保持冷静：不因市场情绪冲动操作
   - 制定计划：提前制定操作计划
   - 执行纪律：严格执行既定策略

3. 调整准备：
   - 设置止损：严格执行止损纪律
   - 减仓计划：制定减仓计划和时机
   - 现金储备：保留足够应对调整

4. 机会把握：
   - 回调加仓：调整时分批加仓
   - 轮动把握：把握板块轮动机会
   - 新热点：关注新兴热点机会
```

```
风险控制案例：

市场环境：牛市持续上涨，估值过高

风险识别：
1. 估值风险：沪深300 PE 22倍，接近历史高位
2. 情绪风险：投资者情绪过于乐观，贪婪情绪浓厚
3. 调整风险：连续上涨后回调风险增加
4. 流动性风险：成交量放大，但估值过高

1. 估值控制：
   - 评估当前估值：22倍PE，接近历史高位

   - 设置纪律：严格执行，不因情绪改变
   - 定期评估：定期评估市场情绪和估值

3. 调整准备：
   - 止损设置：-5%止损，不破关键支撑位

   - 新热点：关注新兴行业机会

实际执行：
- 预警触发：PE达到23倍，启动预警
- 减仓执行：分批减仓10%，锁定部分收益
- 现金储备：提升至30%，为调整做准备

- 成功规避：规避了后续8%的回调
- 锁定收益：锁定部分收益，降低风险
- 把握机会：回调时把握加仓机会
